# electron + react template

### 프로그램 설명
> React와 electron을 사용한 데스크톱 앱을 만들 수 있도록 해놓은 템플릿 입니다.
<br>

## 실행 설명
* 실행 파일 이름: electron_react_template <br>

개발할 때 실행 순서는 다음과 같습니다.
``` C
  C:\electron_react_template> npm start 
```
<br>
